# Laravel API - Customers & Invoices

A simple **Laravel API** project built for practicing backend development:

- CRUD operations for **customers** and **invoices**  
- Bulk update functionality for invoices  
- Built as a **backend-only API**, no frontend included  
- Uses **Laravel**, **PHP**, and **MySQL**

##  Features
- RESTful API routes for customer and invoice management  
- Bulk updates for invoices  
- JSON responses  
- API tested via Postman or any API client
